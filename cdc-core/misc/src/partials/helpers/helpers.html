JSON Helper

ifEquals

Vergleiche die beiden Werte "string" und "string"

{{#ifEquals 'string' 'string' }} Die beiden Vergleichswerte sind indentisch.
{{/ifEquals}}

Vergleiche die beiden Werte "string" und "test"

{{#ifEquals 'string' 'test' }} Die beiden Vergleichswerte sind indentisch.
{{/ifEquals}}

ifUnequals

Vergleiche die beiden Werte "string" und "string"

{{#ifUnequals 'string' 'string' }} Die beiden Vergleichswerte sind unterschiedlich.
{{/ifUnequals}}

Vergleiche die beiden Werte "string" und "test"

{{#ifUnequals 'string' 'test' }} Die beiden Vergleichswerte sind unterschiedlich.
{{/ifUnequals}}

jsonValue

Suche in JSON Datei nach dem Pfad "components.form.input.image"

{{#jsonValue html_attributes "components.form.input.image" }} Der Pfad "components.form.input.image" kontte im JSON gefunden werden.
{{/jsonValue}}

Suche in JSON Datei nach dem nicht existierenden Pfad "components.form.input.TEST"

{{#jsonValue html_attributes "components.form.input.TEST" }} Der Pfad "components.form.input.TEST" kontte im JSON gefunden werden.
{{/jsonValue}}

array

Erzeuge mit Hilfe des Helpers ein Array Objekt

{{#array "['val1', 'val2', 'val3', 'val4']" }} Array Erzeugung
{{/array}}

contains

Prüfe, ob der String "pattern" im übergebenen Array ['components', 'form', 'input', 'image'] enthalten ist

{{#contains "pattern" ( jsonValue html_attributes ( array "['components', 'form', 'input', 'image']" ) ) }} Das "pattern" Attribute ist vorhanden
{{/contains}}

Prüfe, ob der String "TEST" im übergebenen Array ['components', 'form', 'input', 'image'] enthalten ist

{{#contains "TEST" ( jsonValue html_attributes ( array "['components', 'form', 'input', 'image']" ) ) }} Das "TEST" Attribute ist vorhanden
{{/contains}}

Prüfe, ob der String "pattern" im Rückgabewert des JSONValue Aufrufs enthalten ist

{{#contains "pattern" ( jsonValue html_attributes ( array "['components', 'form', 'input', 'image']" ) ) }} Das "pattern" Attribute ist vorhanden
{{/contains}}
